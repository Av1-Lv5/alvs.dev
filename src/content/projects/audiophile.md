### Overview

Audiophile is a e-commerce application built to explore real-world commerce flows beyond static frontend interactions. The project focuses on clean UI, structured data modeling.

### Key Features

- Manage products via **Sanity CMS**
- Add / remove products from cart
- Checkout flow with price breakdown

### Technical Architecture

- Built with **Next JS**, Typescript, and used **Tailwind CSS** for styling.
- Content managed by **Sanity CMS**
- Deployed on **Netlify**
